# Examples

`illustrative_analysis.json` is a manually authored, abbreviated example of the analyzer
contract. It exists to make the data boundary reviewable without spending API tokens. It is
not labeled as a Claude-produced result and should not be used as an evaluation sample.

Real analyzer outputs are written to the ignored `artifacts/` directory by default. Promote
a carefully reviewed real run into this directory later when the repository has a suitable
licensed reference image.
